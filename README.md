# eoffice web

致力于打造一套在线、多人实时协同办公的系统，支持word，Excel，PPT，UML，markdown的在线编辑、预览、pdf的在线预览。需要结合eoffice-server一起使用

## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### 项目启动

```bash
npm start
```

### 项目构建

```bash
npm run build
```

### 代码格式检查

```bash
npm run lint
```

不合规代码快速修正:

```bash
npm run lint:fix
```
